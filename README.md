## 🧠 Концепция

**MultiTimer Pro** — это не просто таймер. Это система управления временем, позволяющая:

- ✅ Создавать **несколько таймеров одновременно** с произвольными именами.
- ✅ Запускать, ставить на **паузу**, **сбрасывать** и **удалять** любой таймер.
- ✅ Получать **звуковое оповещение** и **всплывающее уведомление** по окончании.
- ✅ **Повторять** таймер автоматически (интервальный режим).
- ✅ **Сохранять** все таймеры в локальный файл для восстановления после перезапуска.
- ✅ Визуальный **прогресс-бар** (в версиях с GUI) или цветной вывод (в консоли).
- ✅ Поддержка **горячих клавиш** для быстрого управления.

## 🚀 Функциональность

| Действие                 | Описание |
|--------------------------|----------|
| `Создать таймер`          | Указать имя, общее время (в секундах или формате ЧЧ:ММ:СС). |
| `Запустить` / `Пауза`     | Управление состоянием каждого таймера. |
| `Сброс`                  | Возврат таймера к исходному значению. |
| `Удалить`                | Полное удаление таймера из списка. |
| `Повтор`                 | Автоматический перезапуск после завершения. |
| `Звук`                   | Встроенный сигнал или пользовательский звуковой файл. |
| `Импорт/Экспорт`         | Сохранение/загрузка списка таймеров (JSON). |

## 📦 Содержимое репозитория

| Файл                  | Язык       | Особенности |

## 🛠️ Как запустить

Каждая версия самодостаточна. Установите нужные зависимости и выполните:

```bash
# Python
pip install playsound
python timer_python.py

# Node.js (Electron)
npm install
npm start

# C++ (Qt)
qmake && make
./timer_cpp

# Java
javac timer_java.java && java timer_java

# C# (WPF)
dotnet run

# Go
go mod tidy
go run timer_go.go

# Rust
cargo build --release && ./target/release/timer_rs
